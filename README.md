# Getting started

## Setup project

Select the correct Node.js version (bigger or equal 22)
`nvm use`

Install the dependencies
`npm ci`

## Running project

Run locally
`npm run dev`

Go to http://localhost:3000

## Automated test

Run locally
`npm run test`
